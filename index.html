AFRICAN MUSIC API

This API will give you information about African music.

Name of song, Name of the muscian and their country. It will even give you a link to their youtube channel so you can listen to more.

HOW TO USE

In the address bar input https://african-music-api.herokuapp.com/api/"african song" if there is a space between the name add it. if you would like to see all the songs in the database /api/all .
